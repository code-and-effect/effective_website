# Effective Website

This is a rails 5 starter website that uses most of the effective_* gems.

## Getting Started

- Install rvm.
- Clone this repo.
- Create a config/database.yml based on the example.
- Create a ~/.env based on the example.

- `bundle`
- `bundle exec rake db:create db:migrate db:seed`
- `rails server`

Good luck!
